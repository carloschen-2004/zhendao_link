# 武当山景区门票预订页面 (Wudang Mountain Ticket Booking)

一个简洁、现代的景区门票预订页面，支持多种票型选择和实时价格计算。

## 功能特点

### 🎫 票型支持
- **成人票 [Adult]**: $38.8
- **青少年票 [Nonage] (6-17岁)**: $19.9
- **老年票 [Elderly] (60-69岁)**: $26.6

### ✨ 交互功能
- 实时数量调整（加减按钮）
- 动态总金额计算
- 购物车添加功能
- 直接购买跳转
- 鼠标悬停效果

### 🎨 设计特点
- 响应式布局（移动端/桌面端自适应）
- 现代化界面设计
- 圆角元素
- 橙色主题色
- 图片展示区

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计（传统 CSS，无框架）
- **JavaScript** - 交互逻辑
- **Font Awesome** - 图标库

## 本地运行

1. 克隆仓库
```bash
git clone https://github.com/carloschen-2004/zhendao_link.git
```

2. 打开 `index.html` 文件
   - 直接双击打开
   - 或使用本地服务器：
     ```bash
     python3 -m http.server 8000
     ```
     然后访问 `http://localhost:8000`
